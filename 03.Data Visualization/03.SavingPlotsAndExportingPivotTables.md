# 03.Saving Plots And Exporting PivotTables.md

## Exporting Pivot Tables

```python
df_pivot = df.pivot(index='year',
                    columns='country',
                    values='population')

# export pivot table
df_pivot.to_excel('pivot_table.xlsx')
```

## Saving Plots as png

```python
import matplotlib.pyplot as plt

# Example: Line Plot
df.plot(kind='line', 
        xlabel='X Label', 
        ylabel='Y Label', 
        title='Title',
        figsize=(10, 5),)

# save plot
plt.savefig('my_test.png')
# show plot
plt.show()
```
