# 03.Saving Plots

## Saving Plots as png

```python
import matplotlib.pyplot as plt

# Example: Line Plot
df.plot(kind='line', 
        xlabel='X Label', 
        ylabel='Y Label', 
        title='Title',
        figsize=(10, 5),)

# save plot
plt.savefig('my_test.png')
# show plot
plt.show()
```
